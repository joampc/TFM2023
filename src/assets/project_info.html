Este proyecto representa la construcción y entrenamiento de un modelo de lenguaje que utiliza una capa de embedding seguida de una capa LSTM y una capa densa para generar secuencias de texto. El objetivo puede ser generar texto autónomamente o predecir la siguiente palabra en una secuencia de texto dada. El modelo se entrena utilizando el optimizador Adam y se mide su desempeño utilizando la pérdida de entropía cruzada categórica dispersa y la métrica de precisión.

Esta web está alojada en servicios web gratuitos (Netlify) y el contenedor con la API que da acceso al modelo entrenado se encuentra alojado en la Nube de Google (GCP). Todo el proyecto se sostiene sobre servicios gratuitos, por lo que debido a las limitaciones de uso que definen los servicios en la nube, puede que no esté disponible en algún momento.
